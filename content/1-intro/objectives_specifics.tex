
\begin{itemize}
    \item Implementar un entorno de simulación realista de estacionamiento utilizando CARLA con vehículos y sensores de cámara.
    \item Desarrollar algoritmos de procesamiento de imágenes para la detección de líneas de marcado de cajones de estacionamiento.
    \item Diseñar e implementar un método basado en RANSAC para estimar homografías que ajusten retículas ideales a las líneas detectadas.
    \item Aplicar transformaciones de proyección perspectiva inversa para obtener coordenadas tridimensionales métricas del entorno de estacionamiento.
    \item Validar la utilidad del sistema mediante su integración como componente de percepción en una aplicación de control autónomo.
\end{itemize}
